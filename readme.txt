Add, manage, import, and display WooCommerce shipment tracking details for customers.

== Description ==

WB Smart Order Tracking for WooCommerce helps store owners add shipment tracking data to orders and present it to customers via My Account, email, and a public tracking form.

Features include:

* Order-level tracking management
* Built-in carrier presets
* Public tracking shortcode
* CSV import workflow
* Optional multi-package tracking

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wb-smart-order-tracking-for-woocommerce` directory, or install via the WordPress plugin screen.
2. Activate the plugin through the `Plugins` screen in WordPress.
3. Configure settings in `WooCommerce > Settings > WB Order Tracking`.
